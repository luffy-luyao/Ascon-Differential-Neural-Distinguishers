## Instructions for Execution 

**Instructions for each of the experiments:**

1. **deep_net.py**: run from terminal using the command *python3 ascon_permutation.py* ,This script calls the *ascon_permutation.py* module to generate data.Neural differential distinguishers for the Ascon-p permutation are constructed using three distinct network architectures, all of which achieve comparable distinguishing efficacy. 

2. **deep_net_external.py**: run from terminal using *python3 deep_net_external.py* ,This script calls the *ascon_permutation_external.py* module to generate data. The influence of external inputs on the distinguishing performance of neural differential distinguishers for the Ascon-p permutation is investigated.

3. **deep_net_internal.py**:run from terminal using *python3 deep_net_internal.py* ,This script calls the *ascon_permutation_internal.py* module to generate data. The influence of internal Substitution-Permutation Network (SPN) structure on the distinguishing performance of neural differential distinguishers for the Ascon-p permutation is explored.

4. **deep_net_hash256.py**: run from terminal using *python3 deep_net_hash256.py* ,This script calls the *ascon_hash256.py* module to generate data. Neural differential distinguishers are constructed under the single-message-block input scenario for Ascon-XOF128 algorithms. 

5. **deep_net_xof128.py**: run from terminal using *python3 deep_net_xof128.py* ,This script calls the *ascon_xof128.py* module to generate data. Neural differential distinguishers are constructed under the single-message-block input scenario for Ascon-XOF128 algorithms. 

6. **ascon_permutation.py**:run from terminal using *python3 ascon_permutation.py* ,the base dataset for ascon permutation can be generated on-the-fly by running the provided script.

7. **ascon_permutation_external.py**:run from terminal using *python3 ascon_permutation_external.py* , the dataset for ascon permutation can be generated with different external inputs on-the-fly by running the provided script.

8. **ascon_permutation_internal.py**:run from terminal using *python3 ascon_permutation_internal.py* , the dataset for ascon permutation can be generated with different internal SSPP network on-the-fly by running the provided script.

9. **ascon_hash256.py**:run from terminal using *python3 ascon_hash256.py* , the base dataset for ascon-hash256 can be generated on-the-fly by running the provided script.

10. **ascon_xof128.py**:run from terminal using *python3 ascon_xof128.py* , the base dataset for ascon-xof128 can be generated on-the-fly by running the provided script.

11. **asconpermutation_simulated_data.csv**:This is a representative simulated dataset generated by ascon_permutation with a dataset size of 10000. The same type of data can be regenerated on-the-fly by running the provided script.”No separate data file is required, as the code fully reproduces all results. 

##Dependencies and system requirements

The Python modules provided depend on each other. Download the full archive to run the experiments.

Requirements for running the main attack: python3, current keras installation, h5py. Tested with the tensorflow backend, but the code should be backend-agnostic. Training and evaluation code have the same requirements. Tested configuration: keras 2.10.0, tensorflow 2.10.0, h5py 3.14.0.
